# sparsh

My personal space on the internet. 

A minimalistic, single-page portfolio built with raw HTML, CSS, and vanilla JavaScript. 

I wanted something clean but slightly dynamic, so I built a custom animated ASCII matrix background using HTML Canvas. As you scroll through the different sections (About, Projects, Skills, Contact), the dark gradient overlay expands and contracts to reveal the background underneath. 

## Running Locally

It's completely static—no build steps or frameworks required. You can just open `index.html` directly in your browser, or spin up a quick local server:

```bash
python3 -m http.server 3000
```

Then visit `http://localhost:3000`.

## Built With
- Vanilla HTML, CSS, and JS
- Custom Canvas ASCII rendering 
- Typography: Geist, Inter, and JetBrains Mono
